VIBEZGRAM

VIBEZGRAM is a full-stack social media web application built to bring together modern social networking features such as real-time chatting, notifications, likes, comments, media sharing, and secure authentication into one platform.

The project was created to gain hands-on experience with real-world frontend and backend development, scalable architecture, authentication systems, cloud media handling, and real-time communication.
